Isolation Transaction Property

  • The system state during concurrent transactions is the same as if the transactions were sequential.
  • There are levels to which your database can isolate its transactions. These are called transaction isolation levels. Each prevents your transaction from a different set of problems
  • Transaction isolation levels are a measure of the extent to which transaction isolation succeeds. They are defined by the presence/absence of the following phenomena
