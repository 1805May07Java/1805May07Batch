Three SQL Concepts

Normalization

Process of organizing a DB into tables & columns.

  • 1NF: All attributes are Atomic values (value cannot be divided further).
  • 2NF: Must be 1NF & no partial dependencies (when a column depends on only part of a PK).
  • 3NF: Must be 2NF & no transitive dependencies (when a column depends on another column that is not the PK.
