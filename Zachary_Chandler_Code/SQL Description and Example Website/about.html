About SQL

SQL or Structured Query Language, is a language dealing with many different facets of databases. From designing the database schema, to updating the tables within the database, to managing user access, SQL covers many topics looked for in language to manipulate a database. It works as a:

  1. Data Control Language
  2. Data Definition Language
  3. Data Manipulation Language (CRUD)
  4. Data Query Language
  5. Transaction Control Language

Every table in SQL is broken up into columns and rows. Column headers define what data can go into each attribute within a row. While each row in a table is considered a single tuple or record of those attributes.

Each row in SQL can be queried by using the appropriate SELECT statement on the table the row is in. If the row matches the WHERE clause in the SELECT statement, it will be added to the result set. An example of a SELECT statement would be: 

	SELECT * FROM customers WHERE email LIKE "%@gmail.com"
