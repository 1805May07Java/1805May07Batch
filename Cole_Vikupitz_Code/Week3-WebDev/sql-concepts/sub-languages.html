Sub-Languages of SQL

The syntax of SQL is organized into different categories known as sub-languages. These sub-languages and their respectively most used commands, are outlined below.

Category Common Commands
Data Control Language (DCL) "GRANT", "REVOKE"
Data Definition Language (DDL) "CREATE", "ALTER", "DROP", "TRUNCATE"
Data Manipulation Language (DML) "INSERT", "UPDATE", "DELETE"
Transaction Control Language (TCL) "COMMIT", "ROLLBACK", "SAVEPOINT", "SET TRANSACTION"
Data Query Language (DQL) "SELECT"
