When nested elements each have an event listener, they cannot occur simultaneously, so we must signify the order in which they occur, which is known as event propagation.
Bubbling: target element ---> root/parent element
Capturing: root/parent element ---> target element
In the document.getElementById('').addEventListener(event, function(), useCapture) the "useCapture" parameter takes in a boolean referring to this propagation value.
True = Capture
False = Bubble
