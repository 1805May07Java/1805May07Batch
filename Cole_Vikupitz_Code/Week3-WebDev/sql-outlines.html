What is SQL?

  • Acronym for "Structured Query Language"
  • Allows us to communicate with relational databases
  • Not case-sensitive ('account', 'ACCOUNT', & 'Account' are all the same)

There are a variety of sub-languages within SQL. They are organized into the following categories outlined below.

Category Common Commands
Data Control Language (DCL)
  • 'GRANT'
  • 'REVOKE'
Data Definition Language (DDL)
  • 'CREATE'
  • 'ALTER'
  • 'DROP'
  • 'TRUNCATE'
Data Manipulation Language (DML)
  • 'INSERT'
  • 'UPDATE'
  • 'DELETE'
Transaction Control Language (TCL)
  • 'COMMIT'
  • 'ROLLBACK'
  • 'SAVEPOINT'
  • 'SET TRANSACTION'
Data Query Language (DQL)
  • 'SELECT'
