What is SQL?

  • Acronym for 'Structured Query Language'
  • Allows us to communicate with relational databases
  • Not case-sensitive ('account', 'ACCOUNT', & 'Account' are all the same)

There are a variety of commands available in SQL. They are typically organized into the following categories outlined below.

Category Common Commands
Data Definition Language (DDL)
  • 'CREATE'
  • 'ALTER'
  • 'DROP'
  • 'TRUNCATE'
  • 'GRANT'
  • 'REVOKE'
Data Manipulation Language (DML)
  • 'SELECT'
  • 'INSERT'
  • 'UPDATE'
  • 'DELETE'
Transaction Control Language (TCL)
  • 'COMMIT'
  • 'ROLLBACK'
  • 'SAVEPOINT'
  • 'SET TRANSACTION'
