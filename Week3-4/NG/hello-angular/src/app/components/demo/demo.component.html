Intro to Data Binding

Angular supports data binding, which is a mechanism for coordinating parts of a template with parts of a component. There is both property and event binding

Event Binding

{{ count }}

Event-binding is the process of linking actions to DOM events. Recall that DOM events can be user interactions like mouse movements, key presses or button clicks, as well as browser actions like load and unload. In Angular, we can bind component functions to these actions using a special syntax referred to as event-binding.

Property Binding

Elements on a page all have pre-defined properties. For example, an element has a src property and input elements have a disabled property. That is actually a different construct. HTML controls and defines an element’s attribute. It is a one-time process. When the page is loaded the value is set. Now any changes to the value are actually controlled in the DOM space; the programmatic representation of the webpage.
